# vacation-app - Frontend (Vue.js)

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Tip
```
개발시에는 Backend, Frontend 실행하여 테스트.
빌드시에는 Backend/public 폴더로 Frontend 빌드 결과물이 나오기 때문에
Backend만 실행하여 확인.
```
